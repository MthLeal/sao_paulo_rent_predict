# São Paulo property rent predict

Project with finality to predict rent of properties in São Paulo based on the size.

Analised to see if there is correlation between size and rent of the property and built a linear regression model to make predictions.


## Data Source

Sao Paulo housing prices dataset on Kaggle: [Access the dataset here](https://www.kaggle.com/datasets/renatosn/sao-paulo-housing-prices).


## Installing Dependencies

To create a virtual environment and install the dependencies of the project, execute the following commands:

Create virtual environment:
```
python -m venv env
```

Activate the virtual environment (Command Prompt):
```
env\Scripts\activate.bat
```

or (Bash / Zsh):
```
source env/bin/activate
```

Install the packages from requirements file:
```
pip install -r requirements.txt
```
